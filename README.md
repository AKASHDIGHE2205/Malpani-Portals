# 🏢 Malpani Portals – Multi-Module Management System

## 🚀 Overview

**Malpani Portals** is a full-stack enterprise application consisting of multiple business modules designed to digitize and streamline operational workflows such as courier tracking, store management, and real estate property management.

The system provides centralized control, role-based access, reporting tools, and efficient data management across different domains.

---

## 🎯 Problem It Solves

Organizations often rely on manual processes or disconnected tools to manage operations like courier tracking, inventory records, and property data.

This platform solves:

* Lack of centralized data management
* Inefficient manual record-keeping
* Poor visibility into operations
* Difficulty in generating reports and insights

---

## 🏗️ Modules

### 📦 1. Post Management System

* Manage **inward & outward courier entries**
* Maintain master data (courier types, categories)
* Generate reports based on:

  * Courier type
  * Date range
  * Inward / Outward status
* Export reports as **CSV / PDF**

---

### 📁 2. Store Management System

* Track **store room file entries and records**
* Maintain structured master data
* Role-based data distribution to users
* Generate and export reports (CSV format)

---

### 🏘️ 3. Real Estate Property Management

* Manage **property sale & purchase records**
* Upload layout media (images/maps)
* Interactive plotting system:

  * Mark plot numbers on layout
  * Toggle plot colors based on availability/status
* Generate reports for property data
